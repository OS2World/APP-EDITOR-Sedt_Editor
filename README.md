# APP-EDITOR-Sedt_Editor
SEDT (VMS EDIT-like) text editor. Sedt is a text editor for the IBM PC and compatibles running DOS or OS/2, DEC Rainbow, ATARI ST, VAX/VMS and VAX/ULTRIX.
